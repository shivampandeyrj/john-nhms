# Integrations

## External APIs & Services
- Google Apps Script (Web App URL endpoint `APPS_SCRIPT_URL`)
- Google Sheets (Used as Database via Apps Script `SHEET_NAME = 'Leads'`)
- GmailApp / MailApp (Used for sending emails in Apps Script)
- Google Calendar (Embed mentioned in `admin.html`)
